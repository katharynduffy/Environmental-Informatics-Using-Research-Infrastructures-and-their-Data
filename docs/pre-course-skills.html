Chapter 1 Pre-Course Skills

We will largely be using the following tools to access, pull, and explore data in this course:

  1. R & Rstudio
  2. Git & GitHub
  3. Markdown & Rmarkdown

As such we will need to install and/or update these tools on your personal computer.

You will need to begin by installing these tools from the hyperlinks above.
